How to install Rachel?

First of all remember that, Rachel can be installed just on linux.

Ok, next. You need to have Git, Python, Pip in your system. After, You need to clone repository and after, go for next step.

git clone https://github.com/BlackIQ/Rachel && cd Rachel

Now you cloned Rachel and it is time to install it.

You have 2 ways.

  1. Create a venv and install it with venv.
  2. Install Rachel libs global.

I will show you both of them.

Install with venv

There is nothing changes with venv. The reason is that you may don't want x version of y lib.

python3 -m venv venv
pip3 install -r requirements.txt

Installing without venv

This is a good way but installing with venv is better.

pip3 install -r requirements.txt

Ok, now it is time to run setup.

As you know, Rachel will be installed in your system, so run setup with sudo command.

sudo python3 setup.py

How to develop and debug Rachel?

Well Rachel debug with a debug.py file.

There are 2 files that Rachel works with them.

  • Core file
  • Commands file

Core file has all classes and functions. Adding new feature or things is in code.

But in Commands file we define that whenever with class of function should run.

For example you add new feature and it is time for debug. How should you debug it?

python3 debug.py

Now you debugged Rachel!
