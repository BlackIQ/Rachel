What is Rachel?

Rachel is an assistant writen in Python.
Now Rachel is in Alpha version and still developing.

Rachel has a core that is as a Python library.
In this case, every update or other things are in core.

In fact Rachel can be installed just on linux distros and Mac systems.
But you can develop the core in every operating systems.

Rachel is working under Neotrinost LLC.
Here we make softwares and things you need!


How to install Rachel?

You can install Rachel so easy!
You can get the code via main repository or get zip file from repository.

Best way is install via Github

$ git clone https://github.com/BlackIQ/Rachel
$ cd Rachel
$ python3 setup.py

After that, you need to run Rachel by $ Rachel command.


What is Rachel core?

Rachel Core is the brain. All the classes and functions are in core.
Rachel core in a python library the makes updates and other things so easier.

You can contribute on Rachel Core too.
It is Open source and the repository is linked in references part.
